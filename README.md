# Design Patterns in Python 🧠🐍

## 📌 About This Repository

This repository is dedicated to learning and understanding **Design Patterns in Software Engineering** using Python.

The goal is to explain each pattern in a simple, practical, and real-world oriented way, especially from a **backend development perspective (Django / FastAPI / scalable systems)**.

---

## 🎯 What You Will Learn

In this repository, we cover:

* Core Design Principles (SOLID basics where needed)
* Creational Patterns

  * Singleton
  * Factory Method
  * Abstract Factory
  * Builder
* Structural Patterns

  * Adapter
  * Decorator
  * Facade
* Behavioral Patterns

  * Strategy ⭐
  * Observer
  * Command
  * State

---

## 🧠 Learning Approach

Each pattern is explained with:

* ✔ Simple explanation (problem → solution)
* ✔ Clean Python implementation
* ✔ Real-world backend example
* ✔ When to use it (and when NOT to use it)
* ✔ Comparison with naive / bad approach
